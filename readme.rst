###################
Sorteo
###################

Es una aplicación realizada con el framework CodeIgniter y el motor de base mysql, debemos cargar los achivos y el script en la base mysql
La aplicación te pedira que ingrese los nombres de equipos, personas o lo que tu desees ingresar como alias, una vez ingresado todos los campos 
el sistema formara de forma aleatoria pares entre los datos ingreados.
*******************
Requisitos
*******************

- Codeigniter
- Mysql

*******************
Server Requirements
*******************

- PHP version >=5.3.3
- Apache >=2.0
